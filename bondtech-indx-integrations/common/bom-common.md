# Common Bill of Materials

Components shared across all INDX printer integrations.

## INDX System Components

| Item | Qty | Source | Notes |
|------|-----|--------|-------|
| INDX Toolhead Carriage | 1 | Bondtech | Core carriage with latch mechanism |
| INDX Tool(s) | 2+ | Bondtech | Number depends on printer and use case |
| Induction Heater Module | 1 | Bondtech | Wireless tool heating |

## Dock Hardware

| Item | Qty per Dock | Notes |
|------|-------------|-------|
| 1515 Aluminum Extrusion | Varies | Dock mounting rail (length depends on number of tools) |
| M3 T-nuts for 1515 | 2+ | Dock-to-extrusion fastening |
| M3x8 SHCS | 2+ | Dock mounting bolts |

## Fasteners (Typical)

| Item | Notes |
|------|-------|
| M3 SHCS assorted (8mm, 12mm, 16mm) | Bracket mounting |
| M3 hex nuts | General fastening |
| M3 heat-set inserts | For 3D printed brackets |
| M5 SHCS (if using 2020 extrusion) | Frame mounting |
| M5 T-nuts for 2020 | Frame mounting |

## Tools and Consumables

| Item | Notes |
|------|-------|
| Hex key set (1.5mm, 2mm, 2.5mm, 3mm) | Assembly |
| Thread locker (blue/medium) | Critical fasteners |
| Calipers | Alignment verification |

## Printer-Specific Items

Each printer integration adds its own BOM for:

- Custom brackets and adapters (3D printed or machined)
- Printer-specific fasteners
- Any frame modifications or additional extrusions

See individual printer directories for complete BOMs.
